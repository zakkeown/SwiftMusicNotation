## Overview

SwiftMusicNotation is a modular Swift library for working with music notation. Whether you're building a music education app, a sheet music viewer, or a full-featured notation editor, this library provides the building blocks you need.

### What You Can Build

- **Music Score Viewers**: Display MusicXML files with professional-quality rendering
- **Music Education Apps**: Show notation with playback synchronization
- **Sheet Music Editors**: Use the core models as a foundation for editing
- **Music Analysis Tools**: Access the structured score data for analysis

### Platform Support

SwiftMusicNotation supports:

- **macOS** 13.0 or later
- **iOS** 16.0 or later
- **visionOS** (via iOS compatibility)

The library uses SwiftUI for cross-platform views, with AppKit (macOS) and UIKit (iOS) support available for more control.
